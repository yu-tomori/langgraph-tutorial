## 参考資料

https://langchain-ai.github.io/langgraph/tutorials/introduction/

## セットアップ

.env ファイルを作成

```
LANGSMITH_TRACING=true
LANGSMITH_ENDPOINT="https://api.smith.langchain.com"
LANGSMITH_API_KEY="xxxx"
LANGSMITH_PROJECT="langsmith-project"
OPENAI_API_KEY="xxx"
```

## 実行

```
python3 main.py
```

or

```
python main.py
```
